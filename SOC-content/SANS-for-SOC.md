# SEC 504: Incident handling and security exploits

## Course description

Students learn how to think like adversary and incident responder

Students learn Windows and Linux attacks, API exploitation, password cracking

Student learn how to dig into roots of attack and respond by implementing multi-layered analysis

## Course sections used metodology

### Incident response and cyber investigations
- Build DIAR (Dynamic approach to incident response) to verify, scope, contain and remediate threats
- Practice live system/network investigations

### Scanning and enumeration attacks
- Identify threats and security gaps by exploring attackers reconnaissance techniques & target enumeration
- SMB, Hayabusa, SIGMA, NAC manipulation

### Password attacks and exploit frameworks
- Password and hashes attacks, O365 attacks
- Metasploit, nmap, netcat

### Web app attacks
- Internal and public facing systems
- Forced browsing, IDOR, Command injection, XSS, SQL injection, API exploitation

### Post-exploitation and AI attacks
- Protection bypass techniques, persistance, data exfiltration
- Lateral movement, hijacking attacks


# SEC 501: Defenders enterprise essentials

## Course description

Students learn how to identify threats by understanding adversary TTP.

Students learn how to build defensible networks.

Students learn how to minimize the impact of attack.

Students learn how to use tools to detect anomalies across systems, 

Students learn how to apply defense metodology


## Course sections used metodology

PREVENT - DETECT - RESPONSE strategy

Establish network security audit against baseline

Step into hardening of internal technologies

Further prevent network security measures (ARP resl, DNS resl, ADDS, segmentation, host discovery/visibility)

Start implementing active defense techniques 

Start searching and hunting for specifics (IoC, IoA, vulnerabilities, penetration testing

Setup basic DFIR mechanisms across enterprise
